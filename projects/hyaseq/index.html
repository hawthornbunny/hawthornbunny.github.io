▶ Commands ▶ Note syntax ▶ Examples 
  • tempo: N
    Set song tempo to N, in bpm (default: 150)
  • seq: NOTE_1 NOTE_2 NOTE_3
    Define a note sequence
  • order: SEQ_A SEQ_B SEQ_C
    Define an ordered list of sequences

Notes are specified as 3-character strings:

  • Character 1: Note name - One of c, C, d, D, e, f, F, g, G, a, A, b, or n for noise. Uppercase note names are sharps.
  • Character 2: Octave/cutoff - 0 to 9 for medium-length notes of octaves 0 to 9. For long notes, replace 0 to 9 with A to J. For short notes, replace 0 to 9 with a to j.
  • Character 3: Modulation/volume - 0 to 9 for medium-volume notes with modulation level 0 to 9. For loud notes, replace 0 to 9 with A to J. For quiet notes, replace 0 to 9 with a to j. If the modulation is 0, this character can be omitted.

To specify a rest note, use --- or --.
