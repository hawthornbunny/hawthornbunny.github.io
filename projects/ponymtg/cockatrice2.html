Get a PonyMTG card data file for Cockatrice

To add a PonyMTG set to Cockatrice, you need to obtain a card data file for that set.

Choose a set from the list below, and click Generate Cockatrice file. This should open a new page containing card data for the chosen set in Cockatrice's XML format. Copy the entire contents of that page into a text editor and save it as a file with the extension .xml (example: a_warm_welcome.xml). You can call the file whatever you like, as long as it ends in .xml.

Note: The generated file may modify the names of cards to ensure uniqueness (usually, by appending a set name or a number to the card name). This is to counter a deficiency in Cockatrice whereby cards with the same name cannot have different images.
