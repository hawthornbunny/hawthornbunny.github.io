+ How to use Cockatrice

A quick guide to Cockatrice

Cockatrice is a free, open-source Magic: the Gathering client which allows you to build decks of virtual cards and play against other people online. It also allows you to import custom, fan-made sets and build decks from those. The following is a guide to help you import sets from PonyMTG into Cockatrice.

How to get Cockatrice

Cockatrice can be downloaded from https://cockatrice.github.io/. It's available for Windows, Mac, and Ubuntu derivatives. Installation instructions are provided on the website.

Using Cockatrice

This isn't intended to be a full guide to Cockatrice, but here's the basics: Cockatrice is two things, a deck builder and a game client. When it starts up, you should see it in deck builder mode. Here, you can search for Magic cards and add them to a virtual deck. When you're done building your deck, save it by going to Deck Editor -> Save deck as....

To actually try out the deck you've made, you need to start a game. If you just want to test out the deck by yourself, you can start a single-player local game by going to Cockatrice -> Start local game..., and selecting "1" for the number of players. This will take you to the deck selection screen. Press the Load local deck button and find the deck file that you saved. You will be shown a preview of what your deck looks like. Finally, click Ready to start. This will open a virtual playmat and allow you to try out your deck.

Adding fan-made custom card sets to Cockatrice

By default, Cockatrice walks you through installing the entire Magic: the Gathering database when you start it for the first time, so you most likely already have a full set of official cards in there. Adding unofficial fan-made sets to this is fairly straightforward.

First, you need to have obtained a .xml file for the set of cards you wish to add to Cockatrice. That's what's in the text box at the top of this page. You need to copy that data into a text editor and save it as a .xml file (you can call it whatever you like, as long as it ends in .xml).

Once you've saved the .xml file, you can then import it into Cockatrice by going to Card Database -> Add custom sets/cards/

If the import succeeds, Cockatrice will ask you if you want to enable the set. Choose yes.

You should now be able to search for the cards you've imported, add them to decks, and play games with them.
